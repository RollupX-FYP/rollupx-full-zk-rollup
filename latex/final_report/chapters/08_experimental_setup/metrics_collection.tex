\section{Metrics Collection}
\label{sec:metrics_collection}
During each benchmark run, the Workload Generator submits a steady-state Poisson arrival stream of synthetic transactions. Telemetry data is recorded locally within the containers and exported as JSON and CSV files to the \texttt{benchmark-suite/metrics/} directory.
