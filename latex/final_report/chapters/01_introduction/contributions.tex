\section{Contributions}
\label{sec:intro_contributions}

The primary contributions of this project include:
\begin{enumerate}
    \item The design and implementation of a modular ZK-rollup benchmarking prototype that strictly separates sequencing, execution, proving, and submission stages.
    \item A controlled experimental framework capable of executing reproducible configuration sweeps to measure throughput, latency, finalization time, and cost metrics.
    \item Empirical analysis identifying the critical throughput--latency--cost trade-offs across different batching parameters and Data Availability (DA) modes.
    \item The demonstration of benchmark-mode parameterized proof delay as a valid methodology for analyzing system-level queueing dynamics independently of cryptographic hardware limits.
\end{enumerate}
