\subsection{Data Availability Modes}
The methodology dictates testing two distinct L1 DA submission modes:
\begin{itemize}
    \item \textbf{Calldata:} Compressing the transaction payload and sending it as standard L1 transaction data.
    \item \textbf{EIP-4844 Blobs:} Formatting the payload into a blob and submitting a blob-carrying transaction, significantly reducing the EVM execution gas cost.
\end{itemize}
