While prior work provides insight into deployed rollups, this project focuses on controlled, configuration-level experimentation. Production systems prioritize reliability, EVM compatibility, and user-facing deployment. In contrast, our prototype prioritizes controlled experimentation, modularity, and observability, enabling repeatable, isolated studies to observe exactly how individual design choices shift the fundamental throughput--latency--cost behavior.
