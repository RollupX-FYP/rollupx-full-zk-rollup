\subsection{Write-Performance Scalability}
Write-performance refers to the network's ability to process and confirm transactions (Throughput and Latency). Existing literature extensively explores:
\begin{itemize}
    \item \textbf{On-chain (Layer-1) scaling:} Modifying consensus mechanisms (e.g., transitioning from Proof-of-Work to Proof-of-Stake or adopting DAG structures) and sharding the network state \cite{zhou2020solutions}.
    \item \textbf{Off-chain (Layer-2) scaling:} Moving execution to state channels, sidechains, or rollups. Rollups, in particular, are favored because they do not require a separate consensus mechanism for security, relying instead on the L1 \cite{tran2023implementation}.
\end{itemize}
