\section{Throughput Results}
\label{sec:throughput_results}
Initial measurements captured the baseline throughput of the system. While the sequencer could ingest transactions at a high rate, the committed throughput (measured at the L1 Contract level) was bounded by batch sizes and the L1 block confirmation time of the local Hardhat node.
\vspace{1em}
\noindent \textbf{Note:} The framework supports this metric, but the inspected repository snapshot does not include a completed run for this configuration.
