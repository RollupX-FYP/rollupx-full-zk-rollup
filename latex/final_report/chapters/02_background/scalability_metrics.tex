\subsection{Scalability Metrics}
When evaluating blockchain performance, several key metrics are considered:
\begin{itemize}
    \item \textbf{Throughput (TPS):} The number of transactions the network can process per second.
    \item \textbf{Latency:} The time required for a transaction to be submitted, processed, and included in a block.
    \item \textbf{Finality (Finalization Time):} The time until a transaction is considered irreversible.
    \item \textbf{Cost per Transaction:} The economic cost (e.g., gas fees) paid by the user to execute the transaction.
    \item \textbf{Storage and DA Footprint:} The amount of data added to the ledger, impacting the storage requirements for full nodes.
\end{itemize}
