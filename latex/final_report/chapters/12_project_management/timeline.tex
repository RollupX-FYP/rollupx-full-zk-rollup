\section{Timeline and Milestones}
\label{sec:timeline}
The project was structured into several key milestones, based on the progress report and project proposals:
\begin{enumerate}
    \item \textbf{Phase 1: Architecture and Design.} Defining the microservice boundaries, gRPC interfaces, and L1 smart contract abstractions.
    \item \textbf{Phase 2: Core Implementation.} Developing the Rust-based Sequencer, Executor, and Submitter components, alongside the Python workload generator.
    \item \textbf{Phase 3: Integration and Orchestration.} Containerizing the stack via Docker Compose and ensuring reliable end-to-end transaction flow.
    \item \textbf{Phase 4: Benchmarking and Analytics.} Executing configuration sweeps, troubleshooting integration bugs (such as L1 RPC rate limits), and developing the data-tools pipeline for metrics analysis.
    \item \textbf{Phase 5: Reporting and Publication.} Synthesizing the empirical findings into the final academic report and conference paper submissions.
\end{enumerate}
