\section{Research Outcomes}
\label{sec:research_outcomes}
The expected and delivered outcomes of this project include:
\begin{itemize}
    \item A fully containerized, modular ZK-rollup benchmarking prototype.
    \item The functional implementation of the sequencer, executor, mock prover, submitter, contracts, and workload tools.
    \item A controlled benchmarking methodology defined via orchestration scripts.
    \item Empirical metrics presented in comprehensive result tables and figures.
    \item Derived lessons regarding the impact of batching, sequencing, DA, and proof-delay on finalization.
    \item A reproducibility guide to allow independent verification.
    \item Honest documentation of the prototype's limitations and areas for future work.
\end{itemize}
