\section{Research Objectives}
\label{sec:research_objectives}
To systematically answer the research questions, the project defines the following specific objectives:

\begin{itemize}
    \item Design the modular architecture.
    \item Implement key prototype components (Sequencer, Executor, Submitter, Contracts).
    \item Support configurable batching and sequencing policies.
    \item Support benchmark-mode proof-delay/prover measurement.
    \item Support DA mode comparison (calldata vs blobs).
    \item Generate controlled synthetic workloads.
    \item Collect throughput, latency, cost, DA, proof-delay, and finalization metrics.
    \item Analyze the observed throughput–latency–cost trade-offs.
    \item Provide reproducible documentation and appendices.
\end{itemize}
