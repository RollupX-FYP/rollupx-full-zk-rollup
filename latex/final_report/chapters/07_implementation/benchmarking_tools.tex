\section{Benchmarking and Data Collection Tools}
\label{sec:benchmarking_tools}
The benchmarking suite is orchestrated via \texttt{run\_matrix.sh} and uses Docker Compose profiles like \texttt{--profile core} and \texttt{--profile bench}. The suite provides an automated environment for configuration-matrix testing.

\begin{itemize}
    \item \textbf{Orchestration:} A suite of bash scripts (e.g., \texttt{run\_experiment.sh} and \texttt{run\_pipeline.sh}) manages the lifecycle of the Docker containers, sets configuration variables via \texttt{envsubst}, and ensures component synchronization. 
\end{itemize}
