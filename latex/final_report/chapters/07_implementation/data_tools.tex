\section{Data Tools}
\label{sec:data_tools}
The Data Tools analytics pipeline processes telemetry data by consuming raw JSON/CSV metrics from \texttt{benchmark-suite/metrics/} and aggregating/analyzing them using Python scripts (such as \texttt{aggregate.py} and \texttt{stats.py}) in the \texttt{data-tools/} directory. This pipeline requires dependencies such as \texttt{pandas} and \texttt{matplotlib}.
